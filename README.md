# AI-Career-Path-Recommender
The AI Career Recommendation Tool is a web-based application designed to help interns discover suitable career paths based on their skills and interests. Users enter their technical skills, personal interests and career preferences and the system analyzes this information to recommend relevant job roles along with personalized learning paths.
